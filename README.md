# Ruzxstore Portfolio

Sebuah project **portofolio web** yang menampilkan profil, pendidikan, skills, dan proyek.  
Dirancang agar mudah dibuka di browser, responsif, dan ringan.

---

## 🛠️ Teknologi

- HTML5  
- CSS3 + Tailwind (jika digunakan)  
- JavaScript  
- Font Awesome (ikon)  
- AOS (Animate on Scroll untuk animasi)

---

## 📂 Isi Repository

- `index.html` → Struktur halaman utama  
- `style.css` → Styling halaman  
- `script.js` → Interaksi & animasi  
- `LICENSE` → MIT License  

---

## 🚀 Cara Menjalankan

1. Clone repository:
   ```bash
   git clone https://github.com/rassakiii025-ui/ruzxstore.git
2. Masuk ke folder project:
   ```bash
   cd ruzxstore
3. Edit index.html sesuai kebutuhanmu.
   > `script.js` dan `style.css` tidak usah diubah agar terhindar dari error.

---

## 🌐 Fitur

Website portofolio ini menampilkan:  

- **Home / Welcome**  
- **Tentang Saya / About**  
- **Pendidikan**  
- **Proyek / Portfolio**  
- **Skills**  
- **Contact / Hubungi**  

Dengan animasi halus menggunakan AOS dan desain responsif.
